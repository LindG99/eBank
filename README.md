# eBank - Digital Banking Application

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

eBank is a modern digital banking web application built with ASP.NET Core MVC and C#. It allows users to manage bank accounts, transfer funds, apply for loans, and monitor their financial overview with interactive dashboards.
